========================
Data Scientist en Python
========================

Repositorio de manuales y recursos del entrenamiento **"Data Scientist en** `Python 3.7`_ -
**Nivel básico"**.

.. warning::
   Si esta buscando los manuales y recursos del entrenamiento **Data Scientist en Python** visite el siguiente enlace:
   `https://entrenamiento-data-scientist-python.readthedocs.io/es/latest/ <https://entrenamiento-data-scientist-python.readthedocs.io/es/latest/>`_.

.. sidebar:: Sobre este entrenamiento

    Para dominar el lenguaje de programación se tiene pensado
    como un entrenamiento de 4 a 5 días para las personas que
    son nuevas usándolo o los que quieren aprender acerca de
    las mejores prácticas actuales del desarrollo en Python.

    La planificación de este entrenamiento se estima en:

    - Un entrenamiento de **nivel básico** (*3 a 4 días*) que
      cubre los *seis (06) capítulos*.

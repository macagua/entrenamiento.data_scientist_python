
# Tablero interactivo con Python – Streamlit

Tablero de ventas integrado en Python y la librería ``Streamlit``
para visualizar datos de Excel.

## Ejecute la aplicación
```Powershell
# terminal limpio
streamlit run app.py

# salir
Ctrl-C
```
